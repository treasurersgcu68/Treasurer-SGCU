CSV export

Export CSV ปิดโครงการ

ใช้ไฟล์จากระบบไปสร้างเอกสารปิดโครงการใน Microsoft Word แบบอัตโนมัติ

1 row = 1 document ใช้ XLSX เป็นหลัก
Data source ฐานข้อมูล ใช้ไฟล์ CSV ที่ export จากระบบ แล้วแปลงเป็น mailmerge.xlsx
Document Word Template ใช้ไฟล์ template.docx ที่ดาวน์โหลดจากปุ่มด้านขวา
  1. 1
    เตรียมไฟล์ 2 ไฟล์ กด ดาวน์โหลด Template เพื่อได้ไฟล์ Word เลือก Export ถึงเดือน ถ้าต้องการจำกัดช่วงเวลา แล้วกด Export CSV ปิดโครงการ
  2. 2
    แปลง CSV เป็น XLSX ใน Excel: เปิด CSV แล้วกด File > Save As > Excel Workbook (*.xlsx) ใน Google Sheets: อัปโหลด CSV แล้วกด File > Download > Microsoft Excel (.xlsx)
  3. 3
    ทำ Mail Merge ใน Word เปิดไฟล์ Template ที่ดาวน์โหลดไว้ ไปที่ Mailings > Start Mail Merge > Letters กด Select Recipients > Use an Existing List แล้วเลือกไฟล์ XLSX
  4. 4
    เลือก Sheet และใส่ Merge Fields เลือก Sheet1$ หรือ sheet ที่มีข้อมูล ติ๊ก First row of data contains column headers ใช้ Insert Merge Field ใส่ช่องข้อมูล เช่น «ฎีกา», «โครงการ», «จำนวน», «บัญชี», «เลขที่»
  5. 5
    Preview และสร้างเอกสารจริง กด Preview Results เพื่อตรวจข้อมูลทีละโครงการ ถ้าถูกต้องแล้วกด Finish & Merge > Edit Individual Documents > All
  6. 6
    Export PDF Word จะสร้างเอกสารใหม่ โดย 1 หน้า = 1 โครงการ กด File > Save As > PDF เพื่อส่งออกไฟล์สุดท้าย
คำสั่งหลักใน Word Mailings > Start Mail Merge > Letters Mailings > Select Recipients > Use an Existing List Mailings > Insert Merge Field Mailings > Preview Results Mailings > Finish & Merge
จุดที่ต้องตรวจซ้ำ แถวแรกของ XLSX ต้องเป็นชื่อคอลัมน์ ห้าม merge cell และควรเก็บข้อมูลไว้ใน sheet เดียว หลังเชื่อมกับ Word แล้ว อย่าย้ายไฟล์ XLSX เพราะ Word จะจำ path ของไฟล์ไว้ Microsoft Support: Mail Merge
Actions เตรียมไฟล์สำหรับ Mail Merge ทำในเว็บหน้านี้ให้ครบก่อน แล้วค่อยไปแปลง XLSX และ merge ใน Word
1. ดาวน์โหลด Word Template ดาวน์โหลด Template ได้ไฟล์ .docx สำหรับใช้เป็น Word Template ของเอกสารปิดโครงการ
2. Export CSV จากระบบ
ค่าเริ่มต้น export ทุกเดือน โดยอ้างอิงกำหนดปิดโครงการ
ได้ไฟล์ .csv ที่มีข้อมูลโครงการ ผู้ตรวจ สถานะปิด งบใช้จริง และข้อมูลบัญชีสำหรับ mail merge
